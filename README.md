## 📽️ Core Features & Polish

### 1. Scroll-Controlled Video Hero (Scrubbing)
* The background video scrubs frame-by-frame tied directly to the user's scroll speed.
* 4 custom content overlays fade-in and float up gracefully using high-end cubic-bezier transition curves (`cubic-bezier(0.16, 1, 0.3, 1)`).

### 2. 📱 Smart Responsive Mobile Mode
* **Clutter-Free Space:** The middle slides are automatically hidden using CSS media queries on narrow viewports to prevent phone screen clutter.
* **Consistent Layouts:** Overlays 1 & 4 sit perfectly and symmetrically at the same bottom baseline (`bottom: 8% !important`) with optimized, compact font sizes.
* **Perfect Breaks:** Custom word-breaking on mobile makes "Let's Create Something Amazing" wrap elegantly into exactly two lines, keeping the glowing amber shimmer on the word `"Amazing."`.
* **Clean Video Visibility:** Diluted horizontal and vertical gradient overlays on mobile to keep the subject's face crystal clear and shadow-free!

### 3. 🤖 Google Gemini 2.5 Flash Chatbot Sidebar
* Accessible via a hovering `💬 Ask AI` button at the bottom-right corner.
* Custom-trained using a highly detailed system instruction dataset containing all BBA-CA & MCA coursework, Ethara AI (SFT & RLHF workflows) internship details, and projects list.
* Built-in typing indicators and **conversation history memory** to maintain seamless context during long chats.

## ⚡ Setup & Run Locally

Follow these quick instructions to spin up the development environment in under a minute:

### 1. Clone the Repository
```bash
git clone https://github.com/BhushanC23/bhushan-portfolio.git
cd bhushan-portfolio
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Environment Variables Setup
Create a `.env` file at the root directory of the project and add your Google AI Studio API key:
```env
VITE_GEMINI_API_KEY=your_free_gemini_api_key_here
```

### 4. Start Development Server
```bash
npm run dev
```
Open [http://localhost:5173/](http://localhost:5173/) on your browser to view the application!
